# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅ Yes    |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Instead, email **security@siddiqur.com** with:

- A description of the vulnerability
- Steps to reproduce it
- The potential impact
- Any suggested fix (optional)

You will receive a response within **72 hours**. If confirmed, we will:

1. Work on a fix immediately
2. Release a patch version
3. Credit you in the changelog (unless you prefer anonymity)

Thank you for helping keep `create-wp-plugin` and its users safe.
